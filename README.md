# ForArchives

A cross-platform application for searching and archiving content from various imageboards.

## Features

- Search across multiple archives simultaneously
- Support for multiple archives including 4plebs, Warosu, and more
- Cross-platform support (Web, Desktop, Mobile)
- Command-line interface for automation
- Customizable search parameters
- Media downloading capabilities
